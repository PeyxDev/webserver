**FileMaster Pro** adalah file manager berbasis web dengan tampilan **dark mode profesional** seperti VS Code. Dilengkapi dengan auto installer untuk VPS Linux.

![FileMaster Pro Screenshot](https://via.placeholder.com/800x400?text=FileMaster+Pro+Screenshot)

## ✨ Fitur Lengkap

| Fitur | Keterangan |
|-------|-------------|
| 📁 **File Management** | Upload, download, delete, rename, create folder |
| ✏️ **Code Editor** | Editor seperti VS Code dengan syntax highlighting |
| 🎨 **Dark Mode** | Tampilan profesional dark mode |
| 🔐 **Authentication** | Sistem login untuk keamanan |
| 🚀 **Quick Access** | Navigasi cepat ke folder penting |
| 📊 **File Statistics** | Menampilkan total file, folder, dan ukuran |
| 🔄 **Auto Install** | Script instalasi otomatis untuk VPS |

## 📋 Prasyarat

- VPS dengan **Ubuntu 20.04/22.04** atau **Debian 11/12**
- **Root access** (sudo)
- **Domain/Subdomain** (contoh: filemanager.domain.com)
- Port **80** terbuka
